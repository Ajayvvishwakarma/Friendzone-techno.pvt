## Troubleshooting

| Issue | Cause | Solution |
|-------|-------|----------|
| Port already in use | Another Node process running | `Get-Process node \| Stop-Process` |
| Console errors | External resources failing | All resources now local |
| Page not loading | Server not running | Start server-new.js |
| API not responding | Backend not running | Start api-server.js |
| Images not loading | Wrong path | All local files now served |
| Styling broken | Cache | Clear browser cache (Ctrl+Shift+Delete) |
| Animations choppy | Old browser | Update to modern browser |
